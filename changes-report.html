Release 0.7.0 – Pending

Type Changes By
Fix file descriptor leak. Fixes 72. Thanks to Laurent Goujon (laurentgo). sergei-ivanov
Support passing options to the Java compiler. Fixes 64. Thanks to Julien Henry (henryju). sergei-ivanov
Allowed more flexibility in specifying input proto files. Versioned file extensions (e.g. .proto2, .proto3) can be used. Fixes 57. Thanks to David Reichert (davidmreichert). sergei-ivanov
Removed common goal properties and logic for generating descriptor sets. Instead, two additional goals were introduced for descriptor set generation. Fixes 54. sergei-ivanov
Removed support for JavaNano generator. Fixes 52. sergei-ivanov
Provided origin information for WinRun4J binaries. Fixes 53. sergei-ivanov
Use pb instead of protobin as file extension for binary descriptors. Fixes 51. Thanks to Cosmin Lehene (clehene). sergei-ivanov
Set clearOutputDirectory parameter to false by default. Fixes 48. sergei-ivanov
Java 1.8 and Maven 3.3.9 are now minimum required versions for running the plugin. Fixes 49. sergei-ivanov
More detailed error message. Fixes 47. Thanks to Cosmin Lehene (clehene). sergei-ivanov
